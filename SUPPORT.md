# Support

Graphite is a pre-1.0 open-source project maintained on a best-effort basis.

Use [GitHub Issues](https://github.com/riken127/graphite/issues) for reproducible bugs and focused
feature requests. Include the Graphite version or commit, JDK version, Neo4j version, minimal model
and query, expected behavior, actual behavior, and a small reproduction when possible.

Check the [troubleshooting guide](docs/troubleshooting.md) and existing issues first. General Java,
Spring, or Neo4j operational support is outside the project's support scope. Never include passwords,
tokens, production data, or unredacted logs. Report security concerns through
[SECURITY.md](SECURITY.md), not a public issue.
